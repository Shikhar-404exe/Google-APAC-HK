# Google Cloud Ecosystem Tech Stack

To ensure massive scalability, global availability, and robust AI integration, the platform relies completely on the Google Cloud stack:

* **Compute:** **Google Cloud Run** – Hosts our containerized Python physics engine/API back-end using stateless, autoscaling architecture.
* **AI Engine:** **Vertex AI (Gemini 1.5 Flash)** – Handles multi-modal understanding of citizen feedback, community forum alerts, and structural coordinate calculations.
* **Data Layer:** **BigQuery** – Acts as the data lakehouse logging environmental sensor telemetry, regional weather APIs, and historical asset allocation.
* **Real-time Streaming:** **Pub/Sub** – Handles ingestion of simulated real-time event updates from community data feeds.
* **Storage & Hosting:** **Google Cloud Storage (GCS)** – Houses static dashboard assets, model training blueprints, and simulation logs.
